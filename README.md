# rabbitmq-MyInfo
# RabbitMq的基础使用方法

## 简介

包括读rabbitMq队列，交换机声明定义，以及队列的**兜底**，队列的**发布订阅模式**进行简单的定义以及配置：

包括当队列发生异常时，转发到死信队列

包括当队列消息转发超时时，转发到死信队列

## 注意！！！

在此项目中，基础配置采用的是死信队列进行兜底和手动应答，这样可以最大程度的防止消息因为服务器宕机等其他因素导致消息丢失，**但是**，这并不代表了完全安全，一样有消息丢失的**风险**！！
